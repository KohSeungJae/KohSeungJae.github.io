프로젝트 3 - 게시판 만들기

프로젝트 2를 진행하면서, 웹 페이지를 만드는 과정에대해 조금더 익숙해졌습니다. 그 과정에서 또다른 웹 페이지를 만들어 보고싶다는 생각이 들어 게시판 만들기를 계획하게 되었습니다. 프로젝트 2에서의 웹페이지와 이번에 계획중인 게시판은 차이가 있습니다. 전자는 미리 입력해 둔 정보를 보여주기만 하는 페이지라면, 후자는 사용자와 사용자간의 소통이 가능해야 합니다. 그렇기 때문에 API 서버가 필요하며 이는 파이썬의 Flask를 사용할 계획입니다. HTML, CSS 또한 더 공부하여 이전보다 완성도있는 페이지를 만들어보고 싶습니다.

프로젝트 4 - 게임만들기

프로젝트 1을 진행하면서, 또다른 게임을 만들어 보고 싶다는 생각이 들어 계획하게 되었습니다. 장르는 카드게임 또는 전략게임으로, 여러가지 요소들을 활용하여 재미있게 진행 할 수 있을 것 같습니다. 언어는 파이썬으로 생각하고 있으며 상황에따라 변경될 수 있습니다. 게임의 인터페이스도 구현할 수 있다면 좋겠지만, 일단 게임의 주 설계를 끝마치는 것을 목표로 할 계획입니다.

프로젝트 5 - 머신러닝 구현

저는 컴퓨터 분야중 인공지능에 관심이 많습니다. 그렇기 떄문에 머신러닝과 딥러닝에도 관심이 있는데요. 그렇기 때문에 지도학습을 통한 머신러닝, 예측모델 등 을 구현해보고 싶습니다. 이러한 프로젝트를 실행 하기 위해서는 머신러닝 라이브러리가 필요한데, 텐서플로우 또는 파이토치를 사용할 계획입니다.
